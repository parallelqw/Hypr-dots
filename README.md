This is a customized dotfiles collection with catppuccin mocha palette.

## Installation

```git clone https://github.com/tempestian/Hypr-dots.git```

```cd Hypr-dots```

```./install.sh```

## Keybinds

Mod = SUPER

| Task | Keybind | 
| ------------- |:-------------:|
| Terminal | Mod + Return |
| App Menu | Mod + D |
| Window Kill | Mod + Q |
| Screenshot | PRINT |
| Floating | Mod + F |
| Full Screen | Alt + F |
